# Dialogue art manifest

The three desktop assets were generated with Codex’s built-in image-generation tool on 2026-07-14 (Europe/Berlin). The three compact variants were generated with the same tool on 2026-07-21 (Europe/Berlin), manually reviewed, and converted locally to WebP for static hosting. They are original project assets; no external game art, named artist style, or source composition was copied.

## Shared constraints

- Historical Florence around 1503; painterly Renaissance medium; either a 16:9 desktop dialogue backdrop or a character-led compact portrait composition.
- No imitation of named games, studios, artists, existing compositions, contemporary celebrities, or recognizable modern people.
- No text, logos, watermarks, borders, UI, or collage.
- The supplied workshop image informed only warm lighting and production quality; it was not edited or recreated.
